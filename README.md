# 🏠 House Price Prediction using Linear Regression

## 📌 Project Overview

This project demonstrates a **Linear Regression model** to predict house prices using the Boston Housing dataset. It is a beginner-friendly machine learning project that shows how features like number of rooms affect housing prices.

---

## 🎯 Objectives

* Understand Linear Regression
* Train a machine learning model
* Predict house prices based on features
* Visualize relationship between variables

---

## 📊 Dataset

This project uses the **Boston Housing Dataset**, which contains information about housing in Boston suburbs.

### Key Features:

* **RM** → Average number of rooms
* **LSTAT** → % of lower status population
* **PTRATIO** → Pupil-teacher ratio
* **MEDV** → Median house price (Target)

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas
* Scikit-learn
* Matplotlib

---

## ⚙️ Workflow

1. Load dataset using `fetch_openml`
2. Convert data into numeric format
3. Split dataset into training and testing sets
4. Train Linear Regression model
5. Make predictions
6. Evaluate model using R2 score
7. Visualize results using a graph
